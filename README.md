# Procesamiento de Lenguajes Naturales

Este proyecto contiene ejemplos básicos de **Procesamiento de Lenguajes Naturales (NLP)** utilizando Python. El objetivo es mostrar diferentes técnicas de preprocesamiento y análisis de texto.

## Contenido

1. **Minúsculas**
   Conversión de texto a minúsculas para normalizar los datos.

2. **Palabras vacías (Stopwords)**
   Eliminación de palabras comunes que no aportan significado relevante al análisis.

3. **Expresiones regulares**
   Uso de expresiones regulares para buscar, filtrar y limpiar texto.

4. **Tokenización**

   * Tokenización de oraciones
   * Tokenización de palabras

5. **Derivación (Stemming)**
   Reducción de palabras a su raíz utilizando algoritmos de stemming.

6. **Lematización**
   Conversión de palabras a su forma base o lema.

7. **Análisis de frecuencia**
   Cálculo y visualización de las palabras más frecuentes mediante gráficos.

## Herramientas utilizadas

* Python
* NLTK
* Regex (re)
* Matplotlib
* Jupyter Notebook

## Objetivo

Comprender los pasos fundamentales del **preprocesamiento de texto** utilizados en proyectos de NLP antes de aplicar modelos de aprendizaje automático.
